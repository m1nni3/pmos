🎯 Summary
Budget Start with calculated (ships today, no DB changes). Add D1 table later when you need category-level P&L.
Routing Use sub-routes. Same effort as building a tab switcher, but you get URL state, browser navigation, and deep links for free.
Charts 4 widgets on the dashboard: Cash Flow line, Budget bar, Maintenance doughnut, Recon doughnut — all use data you already have.
